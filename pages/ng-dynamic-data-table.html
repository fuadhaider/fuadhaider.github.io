Angular Dynamic Data Table


A Dynamic Data Table with all necessary features made without any library.


A Responsive Single Page Application by Angular 6 to display a table containing the 5000 rows of json data loaded from https://jsonplaceholder.typicode.com/photos in a user friendly manner. The user is able to choose between displaying 50, 100, 1000 rows at a time and is able to sort and search the dataset. There data, url's and images in data cells and page navigation at the top right.


The whole application was coded by me.
